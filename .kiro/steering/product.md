# Product Overview

DashBoard is a restaurant ordering and meal management system (点餐系统) that provides comprehensive dish management, ordering operations, order management, and nutritional tracking features.

## Core Features

- **Dish Management**: CRUD operations for dishes with categorization (meat, vegetables, staples)
- **Ordering System**: Customer dish selection, quantity modification, and order submission
- **Order Management**: Admin view of order lists, status updates, and statistical reports
- **Nutritional Tracking**: Daily/weekly/monthly calorie and nutrient intake analysis with health scoring
- **Personalized Recommendations**: AI-powered dish recommendations based on user health data
- **Robotic Arm Integration**: Automated meal pickup using Lebai robotic arm with optional YOLO vision detection
- **AI Health Advisor**: Chat-based health consultation with conversation history

## User Roles

- **Regular Users**: Browse dishes, place orders, view nutrition reports, get personalized recommendations
- **Administrators**: Manage dishes, view all orders, generate reports

## Key Technologies

- Backend: Django (Python)
- Frontend: HTML/CSS/JavaScript with modern UI theme
- Database: MySQL
- Hardware Integration: Lebai SDK for robotic arm control
- Computer Vision: YOLOv11 for dish detection (optional)
