## 设计统一要点

### 头部导航栏规范
```css
.top-header {
    height: 64px;
    background: var(--white);
    backdrop-filter: blur(20px);
    box-shadow: var(--shadow-sm);
    border-bottom: 1px solid var(--gray-200);
}

.header-logo {
    width: 40px;
    height: 40px;
    background: var(--gradient-primary);
    border-radius: var(--radius-lg);
    color: var(--white);
}

.header-icon {
    width: 40px;
    height: 40px;
    background: var(--gray-100);
    border-radius: var(--radius-lg);
    color: var(--gray-600);
}

.header-icon:hover {
    background: var(--primary-pale);
    color: var(--primary-blue);
}
```

### 圆角系统升级
- **小圆角**：`var(--radius-md)` (8px) - 标签、徽章
- **中圆角**：`var(--radius-lg)` (10px) - 图标容器
- **大圆角**：`var(--radius-xl)` (16px) - 按钮
- **超大圆角**：`var(--radius-2xl)` (24px) - 输入框、小卡片
- **巨大圆角**：`var(--radius-3xl)` (32px) - 主要卡片
- **完全圆润**：`var(--radius-full)` (9999px) - 标签、按钮

### 装饰元素
1. **顶部渐变条**：卡片顶部 4px 高的蓝色渐变
2. **底部渐变条**：悬停时从底部滑入的蓝色渐变
3. **装饰圆形**：卡片右上角的半透明圆形装饰
4. **光晕效果**：聚焦时的浅蓝色外发光

### 悬停效果
- **小元素**：`translateY(-2px)` + `scale(1.05)`
- **卡片**：`translateY(-4px)` ~ `translateY(-8px)`
- **边框加粗**：从 4px 到 6px
- **阴影增强**：从 `var(--shadow-md)` 到 `var(--shadow-lg)`

- 删除了页面内的重复 CSS 定义
