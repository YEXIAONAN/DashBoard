## 设计语言统一要点

### 圆角系统
- 小圆角：`var(--radius-lg)` (10px) - 用于小图标
- 中圆角：`var(--radius-xl)` (16px) - 用于按钮
- 大圆角：`var(--radius-2xl)` (24px) - 用于卡片
- 超大圆角：`var(--radius-3xl)` (32px) - 用于主要容器
- 完全圆润：`var(--radius-full)` (9999px) - 用于标签和徽章

### 阴影系统
- 小阴影：`var(--shadow-md)` - 用于小元素
- 中阴影：`var(--shadow-lg)` - 用于卡片
- 大阴影：`var(--shadow-xl)` - 用于悬停效果
- 超大阴影：`var(--shadow-2xl)` - 用于重要元素

### 颜色系统
- 主蓝色：#4A90E2 (专业、可信赖)
- 浅蓝色：#7AB8F5 (友好、轻松)
- 深蓝色：#2E5C8A (稳重、权威)
- 绿色强调：#4CAF50 (健康、积极)
- 白色：#FFFFFF (纯净、简洁)
- 灰色系：#F5F7FA → #2C3E50 (10级渐变)

### 过渡动画
- 统一使用 `var(--transition-base)` (0.3s cubic-bezier)
- 悬停效果：`transform: translateY(-2px ~ -6px)`
- 点击效果：`transform: scale(0.95 ~ 1.1)`
