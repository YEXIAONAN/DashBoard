## 💡 使用建议

### 自定义配色
编辑 `modern-theme.css`:
```css
:root {
    --primary-blue: #4A90E2;  /* 改成你喜欢的颜色 */
}
```

### 调整圆角
```css
:root {
    --radius-2xl: 24px;  /* 调整卡片圆角 */
}
```

### 修改阴影
```css
:root {
    --shadow-md: 0 4px 8px rgba(74, 144, 226, 0.12);
}
```
